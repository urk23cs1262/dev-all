docker build -t flask-container .

docker run -d -p 5000:5000 --name myflask flask-container

docker ps

You should see:

myflask   0.0.0.0:5000->5000

Go to:

http://localhost:5000

Output: Flask app running in Docker!

If Port Already Used

Run:

docker run -d -p 5001:5000 --name myflask flask-container

Then open:

http://localhost:5001
